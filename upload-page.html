marmalade-repo

marmalade-repo accepts submissions of FREE software packages for the Emacs environment.

upload a new package

please ensure your package is GPLed or GPL compatible before uploading to marmalade.

what is a package

The ELPA packages which marmalade accepts come in 2 forms, either:

  • a single .el file with a package header
  • or a .tar file with a pkg.el file

Your uploaded file must conform to one of these 2 types.

licences

Your uploaded file should also carry a licence file for a free software licence (for example, the GPL) or a reference to a licence. marmalade-repo does not try to check licences yet but it will in the future.

other ways to upload

You can also upload a package to maramalade directly from Emacs.

To do this you must install our marmalade-upload package.
